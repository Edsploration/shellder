shellder
========
My own zsh theme!

![](http://i.imgur.com/xZJHgq8.png)

#### Installation
With [zplug][]: *(recommended)*
```zsh
zplug "simnalamburt/shellder"
```

With [zgen][]:
```zsh
zgen load simnalamburt/shellder shellder
```

[zplug]: https://github.com/b4b4r07/
[zgen]: https://github.com/tarjoilija/zgen
